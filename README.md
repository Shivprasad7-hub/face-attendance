🎯 AI Face Recognition Attendance System

A full-stack intelligent attendance system powered by AI-driven face recognition.
Built for accuracy, automation, and real-time performance.

🚀 Overview

This project is a smart attendance management system that uses face recognition to automatically mark attendance in real time. It eliminates manual entry, reduces errors, and provides a seamless experience using modern web technologies and AI.

✨ Key Features
👤 Person Registration – Add and manage individuals
📸 Dataset Capture – Capture face images via webcam
🧠 AI Model Training – Train recognition model on captured data
🎥 Live Face Recognition – Real-time detection using camera
📝 Auto Attendance Logging – Attendance recorded instantly
🖼️ Face Gallery – View stored datasets
📊 Dashboard & Reports – Visual attendance insights
🧠 Tech Stack
Frontend
⚛️ React.js
🎨 Material UI
Backend
🐍 Python Flask
📷 OpenCV
🧠 InsightFace
Database
🗃️ SQLite

⚙️ Installation & Setup
🔧 Backend Setup
cd FACE

# Create virtual environment
python -m venv faceenv

# Activate environment
# Windows
faceenv\Scripts\activate

# Mac/Linux
source faceenv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run server
python api_server.py
⚛️ Frontend Setup
cd face-frontend

# Install dependencies
npm install

# Start development server
npm start
